## Features

- Home Page
- Login Page
- Signup Page
- Users Profile
- My Profile
- Create Post (Image, Text) / Delete Post
- Like / Unlike Post
- One Level of Comments
- JWT based Authentication
- Send Friend Request
- Search User
- Explore Page
- Notification Page
- Settings Page
- Logout



---

## Built with

- React JS (v18.1.0)
- React Router v6
- Context API
- Backend made Using ExpressJS & MongoDB (NodeJS)
